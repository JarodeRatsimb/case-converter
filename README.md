# case-converter

## About

A web app that takes a string and converts it into:
- Lower case
- Upper case
- Proper case
- Sentence case

It has the additional functionally of saving the converted text into a .txt file.

## Tech Stack

- HTML
- JavaScript

## Run

Download the code and double click on index.html.
